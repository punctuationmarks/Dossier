Welcome to my simple CRUD app. I tried making this as vanilla as possible, with mostly Python's Django, CSS3, and JavaScript. I tried to made this this easily customizable, in at least using CSS variables for colors. If you know a bit about web design, you should be able to just use this. If you want, you can clone the code from GitHub and throw it on Heroku for a free web app with little annoyance with deployment. Or throw it on a Linode server for just $5 a month (here's a $20 credit, thanks Corey Schafer) that you have complete control over.
